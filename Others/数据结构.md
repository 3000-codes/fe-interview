# 数据结构

## 1. 一维数据结构（线性结构）

线性结构是最常用的数据结构，特点是数据元素之间存在一对一的线性关系。强调存储与顺序

### 1.1 数组

数组是一种线性表数据结构，它用一组连续的内存空间，来存储一组具有相同类型的数据。

数组定长：数组一旦创建后，大小就不可改变（js引擎会自动扩容）

动态扩容：当数组空间不足时，需要扩容，扩容的策略一般是扩容为原来的2倍

优点：
* 通过下标访问元素，时间复杂度为O(1)，查询速度快
    

缺点：
* 数据量较大时，难以找到一块足够大的连续内存空间
* 插入、删除元素时，需要移动元素，时间复杂度为O(n)
* 数组空间必须是连续的，所以当空间不足时，需要重新分配一块连续的空间，再将原数组拷贝过去，时间复杂度为O(n)
